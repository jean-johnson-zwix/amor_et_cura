# client & case management platform

Mission: lightweight open-source case management web application designed for nonprofit staff that is as easy as Google Sheets with structured data and access controls.
Main features: register client, log services provided.

## problem statement
- prohibitive costs for tools e.g. bonterra apricot
- inefficient workflows: manual data entry and compiling funder reports from raw data
- lost context: high staff turnover

## goal
- core case management: client registration, service logging, scheduling, AI tools
- ai-powered intake: photo-to-intake digitization
- automated case notes & summaries: voice-to-structured notes and AI-generated handoff summaries
- automated reporting: auto-generate funder report from raw data tp match funder templates.
